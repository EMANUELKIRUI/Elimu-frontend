# AfricaSchool / Elimu ERP Frontend

This is a static frontend prototype for the AfricaSchool / Elimu ERP backend specification.

Open `index.html` in a browser to view the dashboard. The interface demonstrates:

- Role-based module visibility
- Subscription-package module gating
- School workspace switching
- Approval queues
- Marks and CBC assessment entry surfaces
- Finance, communication, department, and audit summaries

The current app is dependency-free HTML, CSS, and JavaScript, so it can later be connected to backend endpoints without changing the basic screen structure.
